# Проекты

Используйте этот файл для добавления ваших проектов. Просто добавляйте проекты в формате ниже.

**Как использовать:**
1. Добавьте свои проекты в формате ниже (каждый проект разделен линией `---`)
2. Запустите `обновить_сайт.bat` или `обновить_сайт_python.bat`
3. Откройте `index.html` в браузере

**Формат проекта:**
- `## Название` - заголовок проекта
- `**Год:**` - год создания
- `**Теги:**` - теги через запятую (например: `web, frontend, design`)
- `**Описание:**` - описание проекта
- `**Изображение большое:**` - (опционально) путь к большому изображению
- `**Изображение малое:**` - (опционально) путь к маленькому изображению
- `**Ссылка:**` - (опционально) ссылка на проект
